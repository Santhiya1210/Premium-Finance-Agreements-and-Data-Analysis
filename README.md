# Premium Finance Agreements and Data Analysis
# Project Overview
This project analyzes Premium Finance Agreement data using Excel and Power BI to understand loan performance,borrower behavior, and agreement status.
# Objectives
* Analyze total loan amount and agreement distribution. 
* Study agreement status (Active, Completed, Pending, Defaulted). 
* Identify borrower and finance company patterns.  
* Provide basic business insights.
# Data Source
* GoMask AI Dataset (2025)  
* Domain: Finance / Insurance.
# Tools Used
* Excel – Data cleaning and Pivot Tables.
* Power BI – Data modeling, DAX, and dashboards.
# Data Modeling
* Fact Table: Finance Agreements.  
* Dimension Tables: Borrower, Finance Company, Insurance Company. 
* Star schema–based model for analysis.
# Key Metrics
* Total Borrowers: 419.
* Total Agreements: 420.
* Total Loan Amount: ₹25 Million.  
* Completed Loans: 233.  
* Active Loans: 177.
* Defaulted Loans: 7.
# Key Insights
* Most borrowers have only one agreement. 
* Quarterly payment frequency is most preferred.  
* Loan exposure is diversified across borrowers and companies.  
* Default and pending loans are very low.
# Conclusion
The analysis shows a stable and well-managed premium finance portfolio with strong repayment behavior and low risk.
